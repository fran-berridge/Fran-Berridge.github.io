The Difference between ID and Class

The ID and class selectors are two ways to target HTML elements based on their attributes. They are both used to apply CSS styles to specific elements, but they have some differences in how they work.

ID selectors

An ID selector is a unique identifier of the HTML element to which a particular style must be applied.

It is used only when a single HTML element on the web page must have a specific style. To use an ID selector, you need to add an id attribute to the HTML element, and use a hash (#) followed by the id value in the CSS rule.
For example:

HTML
p id="blue">The second paragraph.

CSS
#blue { color: #1c87c9; }

This will apply the blue color to the paragraph with the id of "blue".

Class selector

A class selector is used when the same style must be applied to multiple HTML elements on the same web page1. To use a class selector, you need to add a class attribute to the HTML elements, and use a dot (.) followed by the class value in the CSS rule. For example:

HTML
h2 class="blue">This is some heading.
p class="blue">The third paragraph.

CSS
.blue { color: #1c87c9; }

HTML
h2 class="blue">This is some heading.
p class="blue">The third paragraph.

CSS
.blue { color: #1c87c9;}
