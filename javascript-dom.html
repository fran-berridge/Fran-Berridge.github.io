Thinking about HTML, CSS & JS as a Metaphor

When thinking about HTML, CSS and Javascript as a metaphor we can take inspiration from Lean methodology and “Kaizen”. If we were thinking about the Toyota Motor Corporation which used the ideas from Kaizen to develop the Toyota Production System which emphasised efficeiency, quality, waste reduction and continuous improvement

    HTML would be used to construct the car skeleton, door panels, wheels, trunks and hoods so the car components.

    CSS would be used to add style and basic function to the car, so the leather seats, wooden dashboard and aluminium rims. Now the car is looks good but is still not drivable.

    Using Javascript the car can be made interactive for the driver with a powerful turbo engine, lighting, speakers, dash and display.

Now the car is ready to be promoted to potential buyers based on its reliable structure, sleek style and interactive customer focused features.

What are Control Flows?

Control flows are essentially doing things in a logical order.

For example, in the above Toyota manufacturing example the production line worker would always put the rubber seals on the door frames prior to the next line worker installing the doors.

Most of the time a program will run in a logical order line by line in the order it was written. That works in most cases but sometimes we want the program to do something else depending on what already happened. One way we make this happen is by using if/else statements a bit like an excel formular.

So, in terms of the Toyota a production line employee would (if) install the rubber seals unless (else), there is a defect with the car frame. If a defect is found, the production line stops immediately, pinpointing the issue before it becomes widespread.

Why would I use a Loop?

A loop is a sequence of instructions for the program to repeat until the conditions specified have been met.

As the control flow reads the program it sees a loop is in place. It will then continue to repeat the code until the condition has been meet. Following our production line example, the line will continue until the car is assembled.

What is the difference between an Arrays and an Objects?

An array object can store multiple items in individual elements. An array is created by assigned special [ ] brackets to a variable. You then add your values such as a list of car models inside the square brackets separated by a comma. Array elements are automated numbered so for the Toyota model values [Hilux, Corolla, Camry] Hilux would be assigned the number 0, Corolla the number 1 and Camry the number 2.

An array offers a simple list of values. Whereas, an object is a container for information and includes a name and its value. Using our car production line again the array would be the Toyota car models and an object for the Toyota Hilux could be the price and stock availability

What is a Function?

A function is a piece of code which when performs a specific task and can be re-used multiple times. Inside the function you write the output you want from the instruction you have given. For example, if we were assembling a car, we might add an input of "fit door" and give back a helpful output of "door assembled" once the task has been performed.



The DOM and how it updates webpages

The DOM is like a bridge between web pages and programming languages (such as JavaScript). It represents an HTML document and allows you to change the document’s structure, style and content. It is a way to interact with a webpage using code.
